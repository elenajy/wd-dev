How I Created My Virtual Machine, Installed Software, and Saved My Ova

First, I cloned the base machine, called MasterCopyMint19.

I then choose to clone the current machine, and then I renamed it.

I selected to clone the current machine and to do a full clone.

I went to the command line and did a couple of things. I changed the password. I did this by going down to the command line and typing in "sudo passwd" and following the instructions. I then installed Apache. I did this by going down to the command line and typing in "sudo apt update", and then "sudo apt install apache2".

I then installed Filezilla by typing into the command line "sudo apt install filezilla.

I then installed Atom by going to atom.io and installing it.

After installing all the software, I saved my OVA to the cloud to have a backup copy

And then I had my own VM!
